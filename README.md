## 影视接口
### 搜索
>`https://tools.eyunzhu.com/api/wxgzh/video?siteId=0&wd=斗罗大陆`
> 1. siteId:采集资源站点id，目前资源站点id为[0,11] 欢迎提交优质的资源站点
> 2. wd:搜索关键词

### 获取播放地址
>`https://tools.eyunzhu.com/api/wxgzh/video/detail?siteId=0&url=%2F%3Fm%3Dvod-detail-id-47670.html`
> 1. siteId:采集资源站点id
> 2. url:搜索接口返回的url
